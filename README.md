Portafolio web personal diseñado y desarrollado desde cero para mostrar mis proyectos, habilidades y experiencia como **Desarrollador Full-Stack**. Construido con un enfoque en el rendimiento, la accesibilidad y una arquitectura escalable.

## ✨ Características Principales

- **Internacionalización (i18n):** Soporte nativo para múltiples idiomas (Español e Inglés) gestionado a través de Context API.
- **Diseño UI/UX Premium:** Interfaz minimalista, modo oscuro nativo y acentos de color personalizados (Purple-600) usando Tailwind CSS.
- **SEO y Rendimiento:** Optimización de imágenes de alta calidad servidas a través de CDN (Cloudinary) y metadatos dinámicos integrados en el App Router de Next.js.
- **Protección Anti-Spam:** Formulario de contacto funcional protegido nativamente con Google reCAPTCHA v2.
- **Responsive Design:** Adaptabilidad perfecta desde dispositivos móviles hasta pantallas ultrawide.

## 🛠️ Stack Tecnológico

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Librería UI:** [React](https://reactjs.org/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Iconos:** [Lucide React](https://lucide.dev/)
- **Seguridad:** Google reCAPTCHA v2
- **Despliegue:** [Vercel](https://vercel.com/)
